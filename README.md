# SRT Maker (SRT Lyrics Studio)

SRT Maker is a powerful, web-based tool designed for creating and refining lyrics and subtitles with high-precision waveform visualization. It features a modern, high-performance interface with real-time audio analysis, section management, and specialized synchronization tools.

## Key Features

- **Waveform Visualization**: Powered by Wavesurfer.js for frame-perfect subtitle alignment.
- **Section Management**: Organize lyrics into sections (Verso, Coro, etc.) with custom color coding and top-aligned markers.
- **SRT & .lyric Support**: Import/Export standard SRT files or use the native `.lyric` format for full project persistence (including section metadata).
- **Dynamic UI**: Responsive design with dark mode, smooth animations, and premium gemstone color palettes.
- **BPM & Magnetic Snap**: Tools for snapping subtitle boundaries to the beat or neighboring clips.
- **Overlay Support**: Manage dual-layer subtitles for complex lyric arrangements.

## Deployment

This project is deployed using GitHub Pages.

To build and deploy:
```bash
npm install
npm run deploy
```

## Technology Stack

- **Vite**: Modern frontend build tool.
- **Wavesurfer.js**: Audio waveform visualization.
- **Vanilla JS & CSS**: High-performance core with modern styling.
